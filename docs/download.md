# Download and Install Firux Linux

Firux Linux offers a streamlined installation process that is easy to follow. Thereâ€™s no need to download a separate installer. Instead, you can install Firux Linux directly from the terminal using Pacman.

## Installation Steps

To install Firux Linux, follow these simple steps:

1. Open a terminal on your current Linux distribution.
2. Run the following command to install Firux Linux:

        sudo pacman -S firux_package

3. Follow the on-screen instructions to complete the installation.
